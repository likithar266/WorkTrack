# Freelancer MERN Project

This is a full-stack MERN (MongoDB, Express, React, Node.js) application for a freelance marketplace.

## Development

To run this project in a development environment, follow these steps:

1.  **Start the server:**
    ```bash
    cd server
    npm install
    npm start
    ```

2.  **Start the client:**
    ```bash
    cd client
    npm install
    npm start
    ```

## Deployment

To deploy this project, follow these steps:

1.  **Build the client:**
    ```bash
    cd client
    npm install
    npm run build
    ```

2.  **Start the server:**
    ```bash
    cd server
    npm install
    npm start
    ```

The server will be running on `http://localhost:6001` (or the port specified by the `PORT` environment variable) and will serve the optimized production build of the client.
